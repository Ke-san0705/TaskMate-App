# TaskMate

TaskMate is a Windows desktop mascot task manager built with Electron, React, and Vite. It stays on your desktop, shows today's tasks, supports character images/dialogues, and stores data locally as JSON.

## Requirements

- Windows 11
- Node.js 22 or newer
- npm 10 or newer

## Install

```powershell
npm install
npm run dev
```

If PowerShell blocks `npm.ps1`, run the commands through npm's `.cmd` shim:

```powershell
npm.cmd install
npm.cmd run dev
```

`npm install` also generates the default character PNG files and tray icon from `generate-character.js`.

## Build And Run

Build the renderer:

```powershell
npm run build
```

Start Electron from the built files:

```powershell
npm run start
```

Create a Windows installer with electron-builder:

```powershell
npm run dist
```

## Project Structure

- `main.js`, `main/`: Electron main process, tray, IPC, task watching, notifications, and window management.
- `renderer/`: React UI, hooks, components, utilities, and styles.
- `Chara/`: Character folders. Each character needs `wait.png`, `click.png`, `alarm.png`, and `dialogues.json`.
- `assets/`: Application assets such as the tray icon.
- `data/`: Local JSON data files. Real `*.json` files are ignored by Git so personal tasks/settings are not published.

## Local Data

On first launch, TaskMate creates these files automatically when they are missing:

- `data/tasks.json`
- `data/settings.json`
- `data/notification-state.json`

Example files are included as references:

- `data/tasks.example.json`
- `data/settings.example.json`
- `data/notification-state.example.json`

In packaged builds, writable data is stored in Electron's `userData` directory instead of the bundled app folder.

## Character Packs

Add a new character by creating a folder under `Chara/`:

```text
Chara/MyCharacter/
  wait.png
  click.png
  alarm.png
  dialogues.json
```

`dialogues.json` can define these message groups:

- `click`
- `noTasks`
- `oneTask`
- `multipleTasks`
- `deadlineNear`
- `deadlineNow`
- `overdue`
- `loadError`

Messages may use placeholders such as `{title}`, `{minutes}`, `{count}`, `{time}`, and `{genre}`.

The bundled `Chara1`, `Chara2`, and tray icon PNGs are generated by:

```powershell
npm run generate-assets
```

## Notes

- The app is designed mainly for Windows desktop use.
- `node_modules/`, `dist/`, installer outputs, and personal data JSON files are intentionally excluded from Git.
- After downloading the repository, run `npm install` before starting the app.
